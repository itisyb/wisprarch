# create-wisprarch

Interactive installer for [WisprArch](https://github.com/itisyb/wisprarch) - Fast speech-to-text for Arch Linux with Hyprland.

## Quick Start

```bash
npx create-wisprarch
```

That's it! The interactive installer will guide you through:

1. **Provider Selection** - Choose between Groq (fast, cloud), OpenAI, or local models
2. **Configuration** - Set up your API keys, language preferences
3. **Keybind Setup** - Configure your preferred hotkey (Super+Shift+R by default)
4. **Waybar Integration** - Optional status indicator in your bar
5. **Binary Installation** - Downloads and installs the latest WisprArch binary

## What is WisprArch?

WisprArch is a speech-to-text tool designed for Arch Linux and Hyprland. Press a hotkey, speak, and your words are instantly transcribed and pasted wherever you need them.
